> A great app that allows the user to search for Pc components and save them for future reference! With a polished UI and mobile-first approach, this app is very easy to move around and great to use! This app was built using the MERN stack. With the help of JWTs, CRUD, and Npm tools/react tools. Also, done with Redux and it's react redux toolkit. With the help of Axios, users can get the data they're looking for.
